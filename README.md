Overview

This project is a research-only web-based tool designed to analyze publicly available TikTok content.

The purpose of the project is to study high-level content trends, thematic patterns, and engagement dynamics around selected topics such as lifestyle, food, and consumer culture.

⸻

How the project works
	1.	The system accesses public TikTok video metadata related to specific keywords or topics.
	2.	Data is processed server-side for aggregated analytical purposes only.
	3.	The output consists of high-level insights, such as:
	•	topic trends
	•	content frequency over time
	•	aggregated engagement patterns

No individual-level analysis is performed.

⸻

Data usage & privacy
	•	Only publicly available content is analyzed
	•	No private accounts are accessed
	•	No personal user data is collected or stored
	•	No user profiling or re-identification is performed
	•	All results are anonymized and aggregated

⸻

TikTok integration
	•	The project does not require TikTok user authentication
	•	No content is posted, modified, or managed on behalf of TikTok users
	•	The integration is read-only and used solely for research purposes
